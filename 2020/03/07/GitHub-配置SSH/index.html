SSH和HTTPS克隆仓库的区别

  1. 配置好使用ssh克隆或者更新个人仓库时，不用验证用户名和密码
  2. HTTPS每次操作是需要验证的，但是由于它不用配置，对于新手是非常友好的，直接复制url到git命令行就可完成克隆

准备工作

  1. GitHub账号，如果没有，可以参照我的文章《GitHub注册》，进行注册
